# Peng Pong — Sega Master System Port

A port of the MSX beach volleyball game **Peng Pong** to the **Sega Master System**.

## Original Credits
- **Game concept & code**: Aoineko / Pixel Phenix — licensed CC BY-SA 4.0
- **Sprites**: GrafxKid (CC-BY) — https://opengameart.org/content/super-random-sprites  
- **Graphics**: Yaz
- **Music**: Makoto

## Port Notes
- SMS port retains the full original game logic (Q4.4 fixed-point physics, AI, rules)
- Sprites converted from MSX 2-colour-layer format to SMS 4bpp planar tiles
- Player 2 sprites pre-mirrored (faces left, toward net)
- AY-3-8910 music replaced with SN76489 PSG sound effects
- Keyboard/joystick inputs mapped to SMS joypad (Port A / Port B)

## Controls
| Action          | Port A (P1)       | Port B (P2)     |
|-----------------|-------------------|-----------------|
| Move left/right | Left / Right      | Left / Right    |
| Jump / Serve    | Button 1, 2, Up   | Button 1, 2, Up |
| Menu navigate   | Up / Down         |                 |
| Menu select     | Button 1          |                 |

## Building from Source

Requires **SDCC** (version 4.x) and **Python 3**.

```
make
```

Produces `pengpong_sms.sms` — load in any SMS emulator (e.g. Fusion, Meka, OpenMSX SMS mode).

## File Structure
```
main.c            — Game source (C89, SDCC-compatible)
smslib_sprites.c  — SMS VDP sprite/tile routines
gfx_data.h        — Tile + palette data (generated by convert.py)
SMSlib.h          — devkitSMS header
SMSlib_base.rel   — devkitSMS pre-compiled library (subset)
crt0_sms.rel      — SMS startup code
Makefile
tools/ihx2sms.py  — Intel HEX → SMS ROM converter
```
